# E-Commerce Sales Dashboard | Power BI

![E-Commerce Dashboard](E-Commerce%20Dashboard%20Screenshot.png)

### 📊 Project Overview
This interactive **E-Commerce Dashboard** provides a comprehensive view of sales performance, profit analysis, customer behavior, and regional trends. Built in Power BI, it helps stakeholders make data-driven decisions with clear visualizations and real-time insights.

### ✨ Key Features & Insights

- **KPI Cards**: Total Rate (133M), Total Quantity (275K), Total Purchase Cost (275M), and **Total Profit (454M)**
- **Product Performance**: Bar chart showing profit by product names (Blender, Perfume, Jacket, etc.)
- **Geographical Analysis**: Interactive map highlighting profit across major cities in India and neighboring countries
- **Profit Trend by City**: Area chart showing profit performance across different cities
- **Payment Method Analysis**: Donut chart displaying profit distribution by payment methods (Net Banking, UPI, COD, Cards, Wallet)
- **Regional Performance**: Horizontal bar chart comparing Sum of Rate across West, Central, East, South, and North regions

### How to View
1. Download the `.pbix` file
2. Open it using **Power BI Desktop** (Free)
3. Use the slicers (Payment Method, Gender, Product Category) to interact with the dashboard
